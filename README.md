# DSCI_575_project_shrijaa_shruti
DSCI 575 Milestone 1 project


## Developer Setup

### Dependencies

-   `conda` (version 26.1.0 or higher)
-   Python and packages listed in [`requirements.txt`](requirements.txt)

1.  Install [`conda`](https://docs.conda.io/projects/conda/en/latest/user-guide/install/index.html) as a prerequisite.

2.  Open terminal and run the following commands.

3.  Clone the repository:

    ```bash
    git clone https://github.com/UBC-MDS/DSCI_575_project_shrijaa_csasi.git
    cd DSCI_575_project_shrijaa_csasi.git
    ```

4.  Create and activate the conda environment:

    ```bash
    conda env create -f environment.yml
    conda activate search-app
    ```
